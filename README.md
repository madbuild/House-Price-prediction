# House-Price-prediction
House price prediction using regression models and data analysis.


This project uses machine learning to predict house prices based on a dataset of housing features.  
It includes a Jupyter Notebook for the full workflow and a CSV file with the data.

. About the project
- Loads housing data from CSV  
- Cleans and preprocesses the dataset  
- Trains regression models to predict prices  
- Evaluates performance with metrics and visualizations  

The goal is to provide a clear, reproducible example of how regression models can be applied to real estate data.
